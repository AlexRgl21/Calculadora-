# Calculadora-
Proyecto Optativa
Este proyecto consiste en una calculadora multifuncional desarrollada en PCint, diseñada para reforzar habilidades de programación estructurada, documentación profesional y control de versiones.
La aplicación permite realizar operaciones básicas, cálculos geométricos, estadística básica y generación de la sucesión de Fibonacci, integrando también validaciones para evitar errores comunes.
